Het binaire stelsel is een getalsysteem met alleen 2 cijfers (de 0 en de 1). Een cijfer, ook wel een bit genoemd, staat voor een macht van 2. Voorbeeld: Het binaire getal 1011 omzetten: (1 x 2³) + (0 x 2²) + (1 x 2¹) + (1 x 2⁰) = (1 x 8) + (0 x 4) + (1 x 2) + (1 x 1) = 8 + 0 + 2 + 1 = 11. Waarom het Binaire stelsel belangrijk is? Computers: Het Binaire stelsel is de basis van computertechnologie, info wordt opgeslagen als bits. Homepagina
